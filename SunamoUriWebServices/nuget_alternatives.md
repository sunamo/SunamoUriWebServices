# NuGet Alternatives to SunamoUriWebServices

This document lists popular NuGet packages that provide similar functionality to SunamoUriWebServices.

## Overview

Web service URI utilities

## Alternative Packages

### Flurl
- **NuGet**: Flurl
- **Purpose**: URL building for APIs
- **Key Features**: Query parameters, path segments

### RestSharp
- **NuGet**: RestSharp
- **Purpose**: REST client
- **Key Features**: Request building, URL templates

### Refit
- **NuGet**: Refit
- **Purpose**: Type-safe REST client
- **Key Features**: Automatic URL construction from interfaces

## Comparison Notes

Flurl for manual URL building. Refit for type-safe API clients.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **Flurl**: URL building for APIs
- **RestSharp**: REST client
- **Refit**: Type-safe REST client
